# myapp
## Node+Express+MySQL多表增删查改
- 1.安装NodeJS,npm,MySQL,Navicat for MySQL
- 2.全局安装express:```npm install express express-generator -g```
- 3.开启MySQL服务，打开Navicat，新建连接(连接名任意)，
    <br>再新建数据库，库名为oa(设置编码格式为uft-8)，</br>
